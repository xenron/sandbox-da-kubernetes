Getting Started with Kubernetes - Code Bundle
---------------------------------------------

This code bundle contains examples from the book "Getting Started with Kubernetes". 

Folders named 0X_example contain the examples from the corresponding chapter X.

The docker-image-source folder contains source files for building the docker images used in the book.

This code is an export of the git repo at:
https://github.com/jonbaierCTP/getting-started-with-kubernetes
